# ApexPlanet Task-1: PHP MySQLi Authentication System

**Live Demo:** https://4272fa15-b69b-4a35-aeb9-2cfda84716be-00-3a3t46c5huqi0.pike.replit.dev/register.php

### Features
- User Registration with bcrypt password hashing
- Secure Login with PHP Sessions
- Protected Dashboard page
- Role-based access: user/admin
- MySQLi Prepared Statements for SQL injection prevention
- Logout functionality

### Tech Stack
PHP 8.2, MySQLi, HTML5, CSS3

### How to Run
1. Import `setup.sql` in MySQL
2. Update DB credentials in `db_connect.php`
3. Host on PHP server and open `register.php`
